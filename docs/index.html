Installation Guide

Prerequisites

  • Android 10 or higher recommended.
  • Grant required permissions when prompted (notifications, background activity as needed).
Safety & ethics: Use responsibly. Comply with Bumble’s Terms of Service and local laws. Don’t abuse automation or attempt to bypass app protections.

Steps

  1. Download the latest APK from GitHub Releases or build from source.
  2. Enable installs from trusted sources (Settings > Security > Install unknown apps).
  3. Open the app and review preferences (swipe filters, reply templates, schedule blocks).

Basic Config (JSON)

{
  "filters": { "age": [24, 34], "distance_km": 15, "interests": ["hiking","music","coffee"] },
  "swipe":   { "pattern": "mixed", "random_jitter_ms": [400, 1200] },
  "replies": [
    { "trigger": "hey", "template": "Hey {first_name}! Big plans this week?" },
    { "trigger": "hobby", "template": "Nice! I'm into {interest}. What got you into it?" }
  ],
  "schedule": [
    { "days": ["Fri","Sat"], "start": "19:00", "end": "22:30" },
    { "days": ["Sun"], "start": "11:00", "end": "13:00" }
  ],
  "profile_refresh_hours": 36
}

FAQ

Is it safe on Android?
The app is lightweight and battery-aware. Use conservative settings and follow all platform rules. We do not encourage bypassing protections.
Can I customize reply templates?
Yes—use placeholders like {first_name}. Keep messages friendly and human.
How do schedules work?
Define time windows and days. The scheduler queues actions during the window and applies retry/backoff when needed.
Battery/data usage?
Runs background tasks sparingly and respects Doze modes. You can limit network usage by reducing media and intervals.
Troubleshooting (login, background limits, notifications)
Ensure notifications are allowed, disable aggressive battery optimizations for the app, and verify your Android account permissions.
